# Blink Raspberry Pico

Raspberry Pi Picoを使用したLEDブリンクプロジェクトです。

## 概要

このリポジトリは、Raspberry Pi PicoでLEDを点滅させる基本的なプログラムを含んでいます。組み込みシステム開発の入門として最適です。

## 必要なもの

- Raspberry Pi Pico
- LED
- 抵抗器（適切な値）
- ブレッドボード
- ジャンパーワイヤー

## 使い方

1. 必要な部品を配線する
2. プログラムをRaspberry Pi Picoに書き込む
3. LEDが点滅することを確認する
